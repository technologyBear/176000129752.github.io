官网指标说明

# 基础指标

# 1. 浏览量

网站用户访问的网页次数；用户每打开或刷新一个页面就记录1次，多次打开或刷新同一页面则浏览量累计。

# 2. 访客数

访问网站的不重复用户数（以浏览器cookie为依据），同一访客多次访问网站只被计算1次。

# 3. 登录用户数

周期内访问网站且登录过的不重复用户数。

# 4. 新访客

历史上首次访问网站的访客为新独立访客。

# 5. 页面停留时长（分钟）

访客打开页面到离开该页面的时间差。

# 6. 平均页面停留时长（分钟）

平均每个页面的停留时长，平均页面停留时长=页面停留时长/浏览量。

# 7. 访问量

访客从进入网站到离开网站的一系列活动记为一次访问，也称会话(Session),1次访问(会话)可能包含多个PV。

# 8. 访问时长

访问(会话)中最后1个页面的打开时间与访问开始时间之差。体现网站对访客的吸引程度。

# 9. 平均访问时长

平均每次访问(会话)在网站上的停留时间。平均访问时长=访问时长/访问次数。体现网站对访客的吸引程度。

# 10. 进入次数

该页面作为访问(会话)中的第一个页面的次数。

# 11. 平均进入页面浏览量

进入该页面的会话内浏览次数之和/进入次数。

# 12. 跳出次数

跳出指仅浏览了1个页面就离开网站的访问(会话)行为。跳出次数越多则访客对网站兴趣越低或站内入口质量越差。

# 13. 跳出率

仅浏览了1个页面就离开网站的访问(会话)次数占总访问次数的比率。跳出率=跳出次数/访问量。

# 转化指标

# 1. 注册转化（漏斗）

注册前用户访问过符合规则的页面都计算，注册平台：PC、M。

# 2. 注册转化（最后一次）

只计算注册前最后一个符合规则的页面（非passport.dongao.com），注册平台：PC、M

# 3. 站内方案到达（课程）

最后一次通过站内（refer为dongao.com）跳转到课程方案页面的访客

# 4. 站内方案到达（图书）

最后一次通过站内（refer为dongao.com）跳转到图书方案页面的访客
