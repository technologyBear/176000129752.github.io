活动分析

# 基础指标

# 1. 浏览量

对活动的页面的浏览量。

# 2. 访客

对活动的页面有过访问的用户数量。

# 3.新访客

首次进入网站，且对活动的页面有过访问的用户数量。

# 4.注册人数

注册人数包括两部分： 1、注册前当天最后一个页面是活动配置的页面，归因于该活动。 2、注册的活动码是该活动。

# 5. 活动参与人数

在活动中进行抽奖、领券、秒杀、加购的用户，即为是活动参与人数。

如果一个商品同一时间被多个活动引用，统计的数据不准确。建议一个活动使用独立的商品。

# 6.活动商品支付成功人数

对活动商品有过支付成功行为的用户数量。
